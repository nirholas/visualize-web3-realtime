# Task 15: /agents Page Assembly

## Goal
Assemble the `/agents` page with sidebar, stats bar, live feed, and 3D graph integration.

## Requirements
- Responsive layout with collapsible sidebar, timeline, live feed
- Agent list with status indicators in sidebar
- Tool category filter toggles
- Stats bar with animated counters
- Live event feed with color-coded entries
- Play/pause timeline playback
- Dark/light theme toggle
- Screenshot download
- Deep link support (?agentId=X)
- Keyboard shortcuts (Space, Escape, Arrows, F, L, 1-9)
- "Run Demo" button for mock data
